# 📺 IPTV Гид

Android-приложение для просмотра программы передач (EPG) IPTV-каналов.

## 📱 Возможности

- Загрузка M3U/M3U8 плейлистов по URL
- Группировка каналов по категориям
- Поиск по каналам в реальном времени
- Просмотр программы передач (EPG) с бейджем «Сейчас»
- Прогресс-бар текущей передачи на карточке канала
- Логотипы каналов (загружаются автоматически)
- Избранные каналы (сохраняются между сессиями)
- Открытие канала в VLC одним нажатием
- Поддержка HTTP и HTTPS плейлистов
- Светлая и тёмная тема

## 🛠 Технологии

- **Kotlin**
- **MVVM** (ViewModel + LiveData)
- **Material Design 3**
- **Glide** — загрузка логотипов
- **DrawerLayout** — боковое меню
- **RecyclerView** — список каналов и EPG
- **XMLTV** — парсинг EPG (`.xml`, `.xml.gz`)

## 📋 Требования

- Android 5.0+ (API 21)
- Интернет-соединение
- VLC для Android (для воспроизведения каналов)

## 🚀 Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/7adodjj/IPTVGid.git
   ```
2. Откройте проект в **Android Studio**
3. Нажмите **Sync Now**
4. Запустите на устройстве или эмуляторе

## 📖 Использование

1. Введите URL вашего M3U плейлиста
2. Нажмите **Загрузить**
3. Выберите канал — справа появится программа передач
4. Нажмите ▶ для открытия в VLC
5. Нажмите ☆ чтобы добавить канал в избранное

## 📂 Структура проекта

```
app/src/main/java/com/iptv/gid/
├── data/
│   ├── Channel.kt          — модель канала
│   ├── ChannelGroup.kt     — модель группы
│   ├── EpgProgram.kt       — модель передачи
│   ├── EpgRepository.kt    — загрузка и хранение EPG
│   ├── FavoritesRepository.kt — избранные каналы
│   └── M3uParser.kt        — парсер M3U плейлистов
├── ui/
│   ├── MainActivity.kt     — главный экран
│   ├── ChannelAdapter.kt   — список каналов
│   └── EpgAdapter.kt       — список передач
└── viewmodel/
    └── MainViewModel.kt    — бизнес-логика
```
