# 📺 IPTV Gid

Android-приложение для просмотра IPTV-каналов с поддержкой M3U плейлистов и EPG-программы передач.

## ✨ Возможности

- 📋 Загрузка M3U плейлистов по URL
- 📺 Просмотр каналов через встроенный плеер
- 📅 Программа передач (EPG)
- ⭐ Избранные каналы
- 🔍 Поиск по каналам
- 🌙 Поддержка тёмной темы
- 📱 Минимальная версия Android: 5.0 (API 21)

## 🚀 Установка

### Готовый APK
Скачай последний релиз из раздела [Releases](../../releases) и установи на Android-устройство.

> ⚠️ Перед установкой включи «Установка из неизвестных источников» в настройках Android.

### Сборка из исходников
1. Установи [Android Studio](https://developer.android.com/studio)
2. Клонируй репозиторий:
   ```bash
   git clone https://github.com/ВАШ_НИКНЕЙМ/IPTVGid.git
   ```
3. Открой проект в Android Studio
4. Нажми **Run** или собери APK: `Build → Build APK`

## 🛠️ Технологии

- **Kotlin**
- **MVVM** архитектура (ViewModel + LiveData)
- **Glide** — загрузка логотипов каналов
- **Material Design 3**
- **ViewBinding**

## 📋 Требования

- Android 5.0+ (API 21)
- Интернет-соединение
- M3U плейлист
